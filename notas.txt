Express
Es el framework web mas popular de Node.js, proporciona herramientas y una estructura para 
obtener un mejor codigo, permite desarrollar servidores de manera mas facil.

Caracteristias de Express
Routing, enfocado en alto rendimiento, nos permite desarrollar aplicaciones de node.js mas 
rapidamente con codigo mas conciso

REST
Estilo de arquitectura de software para sistemas hipermedia distribuidos como la World Wide Web
Significa: Represetational State Transfer

RESTful API
Es una API basada en REST

Funciones middleware
se ejecutan: despues de recibir una solicitud y antes de enviar una respuesta.
Tienen acceso al objeto de la solicitud, al objeto de la respuesta y al next(),
una funcion que se llama para ejecutar el proximo middleware.
